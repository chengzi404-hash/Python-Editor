## `Translator` `[translator.py:49]`

Singleton obtained via `get_translator()`. Holds the per-language tables
loaded from `data/i18n/locales/<lang>.json`. Missing keys fall back first to
`en_US`, then to the literal key.

### Methods

| Method | Signature | Description |
| --- | --- | --- |
| `set_language` | `(lang: str) -> bool` | Switch locale. Returns `True` if it changed, `False` otherwise (invalid lang, no-op, or re-entry guard). Fires listeners outside the lock. |
| `add_listener` | `(callback: I18nListener) -> None` | Register a callback. Duplicates are ignored. |
| `remove_listener` | `(callback) -> None` | Unregister; silent on missing. |
| `reload` | `() -> None` | Re-read all bundled locale files from disk. |
| `has` | `(key: str, locale: str \| None = None) -> bool` | Whether the key has a translation in `locale` (or current). |
| `translate` | `(key: str, default: str \| None = None, locale: str \| None = None, **kwargs) -> str` | The lookup core. Falls back to `en_US`, then to `default` or the key itself. Applies `str.format(**kwargs)`. Format failures return the unformatted string rather than raising. |

## Module-level shortcut

### `t(key, default=None, **kwargs) -> str` `[translator.py:164]`

```python
t("menu.file.new")                       # "新建" / "New"
t("greeting", name="Alice")              # uses str.format placeholders
t("missing.key", default="(unknown)")    # safe default
```

Always returns a `str`; never `None`.

### `get_translator() -> Translator` `[translator.py:158]`

Returns the module-level singleton. There is exactly one per process; the
editor and plugins share it.

## Locale files

```
data/i18n/locales/
├── zh_CN.json
└── en_US.json
```

Files are flat JSON dictionaries: `{"menu.file.new": "新建", ...}`. After
editing a file, call `get_translator().reload()` to pick up the change
without restarting.

## Marketplace (`modules.i18n.marketplace`)

Re-exported as `modules.i18n.language_marketplace`. Same shape as the
highlight-theme and plugin marketplaces:

```python
from modules.i18n import language_marketplace
m = language_marketplace.get_marketplace()
m.search("chinese")          # list[MarketplaceSearchResult]
m.download(item, target_dir) # installs the locale pack
```

Use `MarketplaceProvider` as the abstract base when implementing a custom
provider (HTTP, GitHub, …).

## Usage pattern

```python
from modules.i18n import t, get_translator

tr = get_translator()
tr.set_language("en_US")

def on_lang_change(lang: str) -> None:
    rebuild_menu()                       # menu labels come from t()

tr.add_listener(on_lang_change)
```
